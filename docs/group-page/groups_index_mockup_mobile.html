📝 Ghi chú thiết kế mobile

Khác biệt so với desktop:

  • Sidebar → Top bar + Bottom nav — top bar chỉ có brand + notification bell, primary nav xuống bottom (Hướng 3: icon-only inactive, pill+label active).
  • Header gọn hơn — bỏ nút "Tạo nhóm" inline, thay bằng FAB floating ở góc phải dưới (chỉ ở state có nhóm — empty state không cần FAB vì đã có 2 button to trong empty hero).
  • Quick action card — input + nút "Vào" trên cùng row thay vì tách riêng. Compact hơn.
  • Group cards — vertical layout thay vì horizontal grid. Avatar 48px (so với 56px desktop), name + role badge cùng dòng, description xuống dòng dưới. Stats grid 3 cột giữ nguyên (compact đủ ở 358px width).
  • Public groups — horizontal scroll-snap thay vì grid. Mỗi card 240px, hint card thứ 2 lú ra mép phải để báo hiệu có thể swipe.
  • Stat label rút gọn: "Thành viên" → "Tviên" để tiết kiệm width.
  • Role badge "Thành viên" → "Tviên" cùng lý do.

Bottom nav (Hướng 3 đã chốt):

  • Inactive: icon 22px, color #6b7280, 44×44 hit target.
  • Active: pill gradient gold, icon + label, padding ngang 16px, gold shadow.
  • 4 mục: Home · Leaderboard · Groups (active) · Profile.

Empty state mobile:

  • Empty hero không có FAB — 2 button full-width đã đủ rõ.
  • Public groups discovery vẫn hiện bên dưới — heading "Hoặc khám phá nhóm" kết nối với hero.

Touch targets: Tất cả interactive elements ≥ 44×44px. Group card chỉ tap toàn card → navigate /groups/:id. Public card có nút Tham gia riêng (44px tall).

Scroll behavior: Page scroll vertical, public groups horizontal nested scroll với scroll-snap. Bottom nav fixed, FAB fixed.
