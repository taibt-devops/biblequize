📝 Ghi chú thiết kế

Cấu trúc 3 sections (luôn cùng thứ tự):
1. Quick Actions Bar — luôn hiện, cho phép join/create nhanh không cần scroll.
2. Nhóm của bạn — empty state nếu = 0, grid cards nếu ≥ 1.
3. Khám phá nhóm công khai — chỉ hiện nếu có public groups; ở empty state thì làm primary discovery.

Group card hierarchy: Avatar (church icon) + name + role badge → stats grid (3 cột) → footer (activity status + group code/rank). Card của leader có gradient overlay nhẹ + badge gold để phân biệt.

Role badges: leader (gold solid) · mod (sky outlined) · member (sage outlined). Match SPEC mục 9.1.3.

Click group card → navigate /groups/:id (giữ nguyên page detail hiện có).

API gap cần thêm (chưa có trong SPEC v3):
• GET /api/me/groups → list nhóm của user, kèm role + stats summary
• GET /api/groups/public → discovery list, sort by activity hoặc member count

Responsive: Grid auto-fill min 340px (my groups) / 280px (public). Mobile sẽ thành 1 cột tự động.
