✅ 3 thiếu sót đã add:
  • User position stat card prominent: rank lớn (#4), delta hạng tuần này (▲ +2), score, distance to top 1 / top 3.
  • Around-me pattern khi user ngoài top 10: hiện top 10 + divider "···" + user row highlighted + 2 neighbors. (Toggle "User ngoài top 10 (#147)" để xem.)
  • Rank deltas trên mỗi row (▲ 2, ▼ 1, —) — cho thấy ai đang đi lên/xuống tuần này. Tăng engagement.

Format countdown đã fix:

  • Trước: 26 ngày : 06 giờ : 00 phút (clunky, dấu cách lung tung)
  • Sau: 26 ngày · 6 giờ (drop minutes vì còn xa, dùng dot separator)
  • Khi <7 ngày: thêm hours. Khi <24h: thêm minutes. Tự động scale theo thời gian.

API endpoints (mostly reuse + 1 new):

  • GET /api/leaderboard/global?period=weekly&scope=global (existing) — thêm scope param
  • GET /api/leaderboard/around-me?period=weekly (existing) — dùng cho state "ngoài top 10"
  • GET /api/me/leaderboard-rank?period=weekly (mới) — trả về rank + delta tuần trước + distances
  • Backend cần thêm tracking previous-week rank để compute delta. Lưu snapshot weekly via cron hoặc compute on-the-fly từ history.

  • Rank delta computation — backend snapshot weekly rank vào table riêng (đơn giản, query nhanh) hay compute từ session history (tốn CPU)? Snapshot tốt hơn cho scale.
