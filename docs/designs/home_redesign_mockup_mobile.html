📝 Ghi chú thiết kế Home mobile

Khác biệt so với desktop:

  • Greeting card vertical layout — avatar+name+tier ở row 1, tier progress bar full-width row 2, 3 stats row 3 (3 cột với divider). Compact ~160px.
  • Daily Challenge stack vertical — icon+title row 1, meta chips wrapping row 2, big CTA full-width row 3, countdown nhỏ phía dưới.
  • Mode cards single column stack — không 2-col grid, mỗi card full-width. Header row có icon + name + status chip; description compact; footer với meta + arrow.
  • Variety modes horizontal scroll — 3 cards 180px width, scroll-snap, hint card thứ 3 lú ra mép phải.
  • Multiplayer + Tournament stack — 2 cards full-width thay vì grid.
  • Verse + Journey stack vertical — không 2-col, verse trước journey sau.
  • Leaderboard + Activity stack — chỉ ở state-active. Mỗi card riêng, full-width.

Mobile-specific tweaks:

  • Tier progress XP label: "3,120 / 5,000 XP · sao 3/5" — giúp user hiểu cả tier XP lẫn star sub-progress trong cùng 1 dòng.
  • Section heading "không tính XP/leaderboard" rút gọn hơn cho variety modes (vs desktop "không ảnh hưởng XP/leaderboard").
  • Mode card description rút gọn 1 dòng (white-space:nowrap + ellipsis) để fit 358px width.
  • Activity items rút gọn: "Daily Challenge (5/5)" → "Daily 5/5", "Tai Bui Thanh hoàn thành" giữ nguyên nhưng time chỉ "15p" thay vì "15 phút".
  • FAB không có ở Home — không có create action chính.

Bottom nav (Hướng 3):

  • Home active pill với label "Trang chủ" — gold gradient, shadow.
  • Leaderboard / Groups / Profile inactive — icon only 22px.

Above-fold check (sau status bar + topbar = ~720px scroll area trừ bottom nav 84px):

  • State new: greeting (~190px) + DC (~270px) = 460px → motivation card lú ra ~1/3.
  • State active: greeting (~190px) + DC (~270px) = 460px → game modes section heading lú ra.
  • Daily Challenge CTA fully visible above fold — primary action accessible immediately.

Total page length:

  • State new: ~1,750px (8 sections, ngắn hơn vì hide missions/leaderboard/activity)
  • State active: ~2,400px (10 sections — full dashboard)
  • Tương đương 2.5 - 3 viewport scroll, chấp nhận được cho mobile dashboard pattern.
