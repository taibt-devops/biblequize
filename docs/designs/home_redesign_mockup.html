Cấu trúc theo priority P0 → P3:

  • P0 — Greeting card: Tier progress thay 5 ⭐ rating bằng segmented bar + 5 milestone dots. Quick stats inline (streak/energy/season points) thay vì cards riêng. Compact ~120px height.
  • P0 — Daily Challenge hero: Bỏ list 5 sách, bỏ "Chỉ hôm nay" badge. Thêm chip mùa Ngũ Tuần ×1.5. Right side là CTA + countdown.
  • P0 — Motivation card (state-new only): "Bước 1: hoàn thành thử thách". Gentle onboarding instead of empty 0/0/0 sections.
  • P1 — Game modes chính: Practice (always on) + Ranked. State-aware: locked với clear CTA "Đạt Người Tìm Kiếm để mở khóa".
  • P2 — Variety modes: 3 cards (Weekly/Mystery/Speed) — KHÔNG tier-locked theo SPEC mục 5.6. Note "không ảnh hưởng XP/leaderboard" nhỏ.
  • P2 — Multiplayer + Tournament: Section riêng "Thi đấu nhóm". Bỏ "Nhóm Giáo Xứ" khỏi đây — đó là Group feature, có nav riêng.
  • P2 — Daily Missions: HIDE cho user mới. Show khi state-active. (Sidebar không còn duplicate.)
  • P3 — Verse + Journey 2-col: Verse move lên cao thấy được. Journey focus 1 sách cụ thể + reward badge cụ thể, không generic 0/66 bar.
  • P3 — Leaderboard + Activity 2-col: Chỉ state-active. State-new ẩn để tránh comparison âm tính.
Δ Diff với design cũ:
  • ❌ Bỏ: 5-star rating UI (confusing), "Chỉ hôm nay" badge, list 5 tên sách trong DC, Nhóm Giáo Xứ trong game modes, sidebar streak+missions duplicate
  • ✅ Thêm: tier segmented bar với milestone dots, motivation card cho user mới, locked state rõ ràng cho mọi mode tier-locked, mùa Ngũ Tuần chip, Journey next milestone cụ thể
  • 📦 Move: Verse từ footer lên giữa page, Daily Missions từ sidebar vào main
  • 🙈 Hide cho user mới: Daily Missions, Leaderboard, Activity (giảm noise 0-state)

API gap (cần thêm hoặc reuse):

  • GET /api/me đã có — extend response thêm currentBook (sách đang ôn) + currentBookProgress
  • GET /api/me/journey/next-milestone — sách kế tiếp cần focus (1 record thay vì 66)
  • GET /api/me/activity-feed?scope=group&limit=4 — hoạt động trong nhóm (state-active)
  • Existing endpoints reuse: ranked-status, daily-challenge, leaderboard/global, tier-progress, daily-missions

Câu hỏi cho Bui:

  • Tier progress segmented bar OK chứ? (5 dots = 5 sub-stars match TierProgressService logic.) Hay bạn muốn visual khác (stepper, percentage only, hexagons...)?
  • Motivation card "Bước 1" cho user mới — có muốn thêm "Bước 2: Đạt Người Tìm Kiếm để mở Ranked" sau khi user complete DC đầu tiên không?
  • Variety modes section — currently note "không ảnh hưởng XP/leaderboard" trong heading. Có rõ ràng đủ không, hay cần icon warning visible hơn?
  • Activity feed (state-active) — backend hiện chưa có, build v1 hay defer?
